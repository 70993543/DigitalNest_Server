# DigitalNest | Proyecto IV Evidencia

---

Clonar repositorio
```cmd
    git clone https://github.com/70993543/DigitalNest_Server.git <renombra si es necesario>
```

Instalar dependencias
```cmd
    npm install
```

Crearse un .env y guiarse del archivo env.example

Iniciar servidor
```cmd
    npm run dev
```
